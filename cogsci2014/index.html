Online Experiments with jsPsych, PsiTurk, and Amazon Mechanical Turk

tutorial at the 2014 Cognitive Science Society annual meeting

Joshua R. de Leeuw, Anna Coenen, Doug Markant, Jay B. Martin, John V. McDonnell, Alexander S. Rich, and Todd M. Gureckis

Before the tutorial, we recommend that you:

  • Install an up-to-date WebKit-compatible browser (such as Google Chrome).
  • Install a programming-friendly text editor, such as Notepad++ (Windows) or TextWrangler (OSX).
  • Download and install PsiTurk, which requires either OSX or another Unix-based system. Windows users can install PsiTurk on a cloud server using OpenShift. Instructions are available in the PsiTurk documentation.
  • Browse the PsiTurk documentation
  • Create an account on psiturk.org
  • Create an Amazon Web Services account and register for Mechanical Turk. Instructions are available at the PsiTurk website.
  • Complete an introductory lesson at Codecademy if you have never programmed with HTML or JavaScript before.

Problems getting set up?

We've set up a Q&A style forum through google groups. Please post questions and problems there so that solutions are available to all.
