Online Experiments with jsPsych, PsiTurk, and Amazon Mechanical Turk

Joshua R. de Leeuw, Anna Coenen, Doug Markant, Jay B. Martin, John V. McDonnell, Alexander S. Rich, and Todd M. Gureckis

tutorial presented at the 2014 Cognitive Science Society annual meeting

Before the tutorial, we recommend that you do a few things to prepare:

  • Install an up-to-date, standards-compliant web browser (such as Google Chrome).
  • Install a programming-friendly text editor, such as Notepad++ (Windows) or TextWrangler (OSX).
  • Install PsiTurk, which requires either OSX or another Unix-based system. You can find instructions for installing PsiTurk at link.
  • Create an Amazon Web Services account and register for Mechanical Turk. Instructions are available at the PsiTurk website.
  • If you have no experience with HTML, CSS, and JavaScript, you might benefit from completing a few lessons at Codecademy.
